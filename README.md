# 📚 课时工资计算器

一个专为教师设计的课时工资记录和计算工具，界面美观，功能实用。

## ✨ 功能特点

- **📝 每日记录** - 快速记录每天的课程、课时和课时费
- **💰 自动计算** - 自动计算每条记录的工资，支持小数课时
- **📊 统计分析** - 实时显示本月和累计的工资、课时统计
- **📅 历史查看** - 按月份筛选查看历史记录
- **💾 数据导出** - 支持导出JSON格式的数据备份
- **🎨 美观界面** - 现代化设计，响应式布局，支持移动端
- **💻 本地存储** - 使用浏览器localStorage，数据安全可靠

## 🚀 使用方法

1. **直接打开** - 双击 `index.html` 文件在浏览器中打开
2. **添加记录** - 填写日期、课程名称、课时数和课时费
3. **查看统计** - 顶部卡片实时显示工资统计
4. **管理记录** - 可以按月份筛选、删除单条或清空所有记录
5. **导出数据** - 点击"导出数据"按钮备份你的记录

## 📋 功能说明

### 添加课时记录
- **日期**：选择上课日期（默认为今天）
- **课程名称**：例如"数学"、"英语"等
- **课时数**：支持0.5的倍数，例如1、1.5、2等
- **课时费**：每小时的费用
- **备注**：可选，记录班级或其他信息

### 统计卡片
- **本月总工资**：当月所有课时的总收入
- **本月总课时**：当月累计的授课时长
- **累计总工资**：所有记录的总收入

### 数据管理
- **月份筛选**：快速查看特定月份的记录
- **导出数据**：将所有记录导出为JSON文件
- **清空数据**：清除所有记录（需二次确认）
- **删除记录**：删除单条记录

## 💾 数据存储

所有数据存储在浏览器的 localStorage 中，即使关闭浏览器也不会丢失。建议定期使用"导出数据"功能备份。

## 🌐 浏览器兼容性

支持所有现代浏览器：
- Chrome / Edge (推荐)
- Firefox
- Safari
- Opera

## 📱 移动端支持

完全响应式设计，可在手机和平板上完美使用。

## 🎯 使用场景

- 📖 记录日常授课情况
- 💼 月底核对工资
- 📈 分析收入趋势
- 🗂️ 整理课时档案

## ⚠️ 注意事项

- 数据存储在浏览器本地，清除浏览器数据会导致记录丢失
- 建议定期导出数据备份
- 切勿在多个浏览器间共享数据（每个浏览器独立存储）
